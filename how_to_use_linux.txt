Linux の基本的なコマンド操作一覧表

- - - - - - - - - - - - - - - - - - - - - — - - - - - - -
ctrl + b カーソル移動(-1)
ctrl + f カーソル移動(+1)
ctrl +a カーソル先頭
ctrl + e カーソル後方移動
option + b　1単語分移動(-)
option + f　1単語分移動(+)
ctrl +d その時カーソルのある位置を削除
delete その時カーソルのある一個前の位置を削除
ctrl +w 1単語分削除する(単語の途中の時はくぎれ目まで）
ctrl + k カーソル位置から行末までの単語を削除する
ctrl + u カーソル位置から行の先頭までを削除する
ctrl +y 最後に削除した内容を復元
(ctrl + u -> ctrl +y で復元可能)

ctrl + c プロセス強制終了
ctrl + l 画面をリセット
l
Tab保管機能(Juypterと同じ)-> 2回押すとそれで始まる単語のリストが出る

ctrl + p 一個前のコマンドに移動
ctrl + n 一つ先のコマンドに移動
ctrl + r コマンド履歴を検索 -> ctrl +g 検索結果を破棄してプロンプトに戻る


- - - - - - - - - - - - - - - - - - - - - — - - - - - - -
pwd カレントディレクトリの取得
相対パス
	. カレントディレクトリ
	.. 親ディレクトリ

cd X:Xにディレクトリを移動(X に何も指定しなければ ホームディレクトリに移動する)
ls ディレクトリ内部のファイルを検索する
	ls X: Xディレクトリの内部に含まれるディレクトリやファイルのリストを出す
	ls X Y: X,Y ディレクトリの情報をそれぞれ表示する
	ls A*:カレントディレクトリにおいてAで始まる任意のディレクトリやファイルを表示する
	ls A?:A+ カレントディレクトリにおいて任意の一文字のディレクトリやファイルを表示する

ls -(文字) のコマンドオプション -> 複数コマンドを並べて描くこともOK
ex.)
	ls -l : ファイル名以外にファイル属性などの情報を表示する
	ls -a : 隠しファイルも表示する
	ls -F : ファイル種類を表示する


- - - - - - - - - - - - - - - - - - - - - — - - - - - - -
mkdir :　新しいディレクトリを作成(既出の場合はerror)

深いディレクトリを一度に作成するには mkdir -p ~~~/~~~/~~~のようにする


touch ファイルを作成する　-> touch <file 1> <file 2>
rm    ファイルを削除する -> rm < file 1> <file 2>


パス名展開を利用して .txt などのファイルをまとめて削除することもできる
ex.)
	rm *.txt

ディレクトリを削除するときは rm -r <dir1> <dir2>.
-> rmdir <dir1> :中にファイルが残っていると削除できない.


 rm -i <file1> <file 2>　-> 削除の確認を表示させることができる


cat <file1> <file2> file1 ,file2…を表示する (内部を見る)
cat -n <file1>… で行番号を表示する

less (option) <file 1> : ファイルの内容をスクロール表示する
-> 
f(1画面分下にスクロール)
b(1画面分上にスクロール)
q(lessコマンド終了)
ファイル内部の検索(ここでは省略。文字列の検索や検索結果の上下移動などを行うことができる)


cp (option) <file1,file2,file3..> <dir>:ファイル(ディレクトリ)たちを新しいdirにコピる
上書きなどに注意
(option) = -i で消去確認できる
ディレクトリのコピーの時は cp -r dir1 dir2 (dir2にdir1をコピる)

mv (option)<file1,file2..> <dir/file_name> :ファイルの移動ないしは名前変更
上書きなどに注意
(option) = -iで消去確認できる

ln (option) <file_1> <file_2> ファイル1のリンクとしてファイル2を作成する
(option) = -s でシンボリックリンクの作成を行う（何も付けない場合はハードリンク)

->深いところの長いパスをもつファイルにも簡単にアクセスできるなどのメリットがある。
